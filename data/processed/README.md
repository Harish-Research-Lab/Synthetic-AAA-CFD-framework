# Processed results and paper-figure map

This folder holds the processed analysis outputs behind the figures in the paper
(RSOS-260428). Every figure below is regenerated by a script in the repository-root
`analysis/` directory; run each script from the repository root with the packages in
`requirements.txt` installed.

## Paper figure → source folder → generating script

| Paper figure(s) | Folder here | Script (`analysis/`) |
|---|---|---|
| 3, 14–17 (parameter distributions) | `distribution_plots/patient_histograms_modified/` | `distribution_with_patient_data_modified.py` |
| 18–21 (Q–Q plots) | `distribution_plots/qq_plots/` | `qq_plots.py` |
| 25–28 (patient vs morphed comparison) | `distribution_plots/multi_comparison_modified/` | `distribution_modified.py` (two-way) |
| 4, 22, 23, 24 (parameter relationships) | `parameter_plots/Manual_modified_plots/` | `data_bound_with_patient_data_manual.py` |
| 5 (convex-hull analysis) | `parameter_plots/Manual_modified_plots/hulls_only/` | `convex_hull_creator.py` |
| 6, 7, 29, 30, 31, 32 (population bounds / universal interior) | `bound_plots/` | `data_bound_with_morphed_data_manual.py` |
| 8 (geometry–hemodynamics correlation) | `correlations/` | `correlations/correlation.py` |
| 11 (cycle convergence) | `cycle_convergence/` | see `cycle_convergence/README.md` |
| 12, 13 (mesh convergence) | `mesh_convergence/` | see `mesh_convergence/README.md` |

`distribution_modified.py` can produce two kinds of comparison plot: **two-way**
(patient data vs. all synthetic cases) and **three-way** (patient data vs. all cases vs.
the "interior" subset). The paper uses the two-way plots, so only those are included here;
the three-way plots are omitted from this release but are regenerated automatically by the
same script when run.

The `bound_plots/` figures (6, 7, 29–32) support six outlier-handling methods
(`dbscan`, `iqr`, `mahalanobis`, `manual`, `no_outliers`, `zscore`). The paper uses only the
manually curated variant, so each group folder here keeps just its `manual_modified/`
subfolder; the other five methods are omitted from this release but are regenerated
automatically by `data_bound_with_morphed_data_manual.py` (and the related bound scripts)
when run.

## Figures not generated by this pipeline

Figures 1 and 2 (anatomical-location and shape-morphing schematics) and figures 9 and 10
(WSS contour and S-shape CFD renders) are produced outside this code base (illustration and
CFD post-processing/ParaView) and are therefore not reproduced here.

## Other files

- `fitted_distributions.json` — fitted marginal distributions per age/sex group (input to the distribution and Q–Q plots).
- `convex_hull_metadata.json` — convex-hull vertices/metadata used by the parameter-space plots.
- `correlations/`, `cycle_convergence/`, `mesh_convergence/` — see the README inside each.
