# Hemodynamic–geometry correlation results

Processed correlation results underlying the paper's hemodynamic analysis, for two inlet velocity boundary conditions: **parabolic** and **plug**. Each script anchors to this folder (`Path(__file__).parent`), so run them from here — no arguments needed.

## Contents

- `parabolic/aaa_parameters_metrics_parabolic.xlsx`, `plug/aaa_parameters_metrics_plug.xlsx` — per-case geometric/anatomical descriptors and extracted hemodynamic biomarkers (TAWSS, OSI, WSS-derived metrics) for each inlet condition. These are the inputs to both scripts.
- `all_correlations_parabolic.csv`, `all_correlations_plug.csv` — full correlation tables (generated by `top_correlations.py`).
- `top_correlations_summary.txt` — ranked strongest correlations (generated by `top_correlations.py`).
- `parabolic/`, `plug/` — correlation-matrix figures: `all_parameters_hemodynamic_correlation.png`, `geometric_hemodynamic_correlation.png`, `anatomical_hemodynamic_correlation.png` (generated by `correlation.py`).

## Scripts

```bash
# from this directory
python correlation.py        # reads the metrics .xlsx, writes correlation-matrix PNGs into parabolic/ and plug/
python top_correlations.py   # reads the metrics .xlsx, writes all_correlations_*.csv and top_correlations_summary.txt
```

Requires the packages in the repository-root `requirements.txt` (pandas, numpy, matplotlib, seaborn, openpyxl).
