# Mesh (grid) convergence results

Processed grid-independence results for the peak-systolic centreline velocity, underlying the
paper's mesh-convergence study. Richardson extrapolation over the three finest grids
(N = 2.75, 4.68, 7.72 M) gives an observed order of accuracy p = 1.94 and a fine-grid
GCI of 2.2 %.

## Contents

- `mesh_convergence.xlsx` — the curated per-grid centreline velocity data (sheet `Sheet5`); this is the input to the plotting script.
- `plots/` — figures generated by the script:
  - `improved_mesh_convergence_plot.png` / `.pdf` — centreline velocity for all grids (paper figure).
  - `mesh_convergence_with_ci.png` / `.pdf` — adopted vs. finest grid with a 95 % bootstrap confidence interval (paper figure; appears in the manuscript renamed as `mesh_convergence_with_ci_combined.pdf`).
  - `improved_mesh_difference_plot.png` / `.pdf` — relative difference between successive grids.
  - `improved_point_convergence.png` / `.pdf` — point-wise convergence.
  - `gci_analysis.png` / `.pdf` — grid convergence index analysis.

## Scripts

```bash
# from this directory
python mesh_convergence_along_line_plot.py   # reads mesh_convergence.xlsx, writes the figures into plots/
```

Requires the packages in the repository-root `requirements.txt` (pandas, numpy, matplotlib, openpyxl).
